# Semáforo Interactivo con React

Este es un proyecto que simula un semáforo funcional utilizando React y hooks para el manejo de estados.

## Características
- **Lógica de estados:** Utilicé el hook useState para gestionar qué luz está encendida y permitir que el usuario cambie el brillo al hacer clic directamente sobre cada círculo.
- **Efectos visuales:** Implementé estilos CSS personalizados para lograr un efecto de iluminación neón que simula el resplandor de las luces en la oscuridad.
- **Funcionalidad extra:**
  - Un botón para alternar automáticamente entre los colores (rojo, verde, amarillo).
  - Un modo especial que añade una cuarta luz de color púrpura al semáforo de forma dinámica.

## Tecnologías utilizadas
- React.js
- JavaScript (JSX)
- CSS3
- Vite
